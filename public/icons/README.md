# PWA Icons

Place your PWA icons here:

- `icon-192x192.png` — 192×192px, brand colour #00AEEF background with "KE" or lightning bolt icon
- `icon-512x512.png` — 512×512px, same design

These are referenced in `/public/manifest.json` and in `app/layout.tsx` (apple-touch-icon).

Quick generation options:
- https://realfavicongenerator.net/ — upload an SVG, download all sizes
- https://maskable.app/ — preview maskable icons for Android
